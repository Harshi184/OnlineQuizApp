Online Quiz Application


Online Quiz Application is a responsive web-based platform that allows users to test their knowledge in Java, Python, C Programming, and Aptitude.

Features

* User Registration (Name Entry)
* Multiple Categories
* 120+ Questions Stored in JSON
* Random Question Generation
* Timer-Based Quiz
* Progress Tracking
* Score Calculation
* Leaderboard
* Answer Review System
* Certificate Generation for 100% Score
* Responsive UI

Technologies Used

* HTML5
* CSS3
* JavaScript
* JSON
* Local Storage
